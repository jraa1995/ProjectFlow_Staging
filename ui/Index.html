Repair Project Owners

Finds projects whose Primary Owner is stored as a name (e.g. "Firstname Lastname") instead of an email, and replaces it with the matching user's email so the edit form pre-selects correctly.

Normalize Project Owners

Sweeps every project. Resolves names to emails when a single match exists, moves unresolved/ambiguous names to pendingOwnerName/pendingSecondaryUsers, and lowercases existing email IDs. Run after a worklog import or a large user-roster change.

Run a dry-run first to enable Apply.
